# アニメの撮影さん2 サポートファイル #

## 作成環境 ##
Windows10  
Adobe AfterEffects CC2017か2018

## 内容物 ##
getFiles.jsx
getShapeProperties.jsx

## getFiels.jsx ##
本文と若干異なっており、フォルダ選択ダイアログが出てフォルダを選択すると、そのフォルダを再帰的に走査していってファイル名を表示していきます。再帰的になっている為、深い階層のあるフォルダを選択すると相当時間が掛かりますので注意してください。

## getShapeProperties.jsx ##
選択したレイヤーのシェイプを再帰的に走査して、表示します。
